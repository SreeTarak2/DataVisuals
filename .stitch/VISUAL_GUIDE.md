## Comparison: Which Color Scheme?

Each loader responds to **4 different stages** with different colors:

```
🔵 ANALYZING (Blue → Cyan)
   "Analyzing your question"
   
🟣 PROCESSING (Purple → Pink)
   "Processing data"
   
🟠 GENERATING (Amber → Orange)
   "Generating insights"
   
🟢 REANALYZING (Emerald → Teal)
   "Refining results"
```

This gives users visual feedback about what stage of the AI pipeline they're in.

## Animation Details - What Makes Them Professional?

### Option 1: 3-Layer Depth
```
Layer 1: Conic-gradient rotating ring (3s)
Layer 2: Pulsing gradient ring (2s)
Layer 3: Floating icon with bob animation (2s)
Result: Deep, sophisticated, premium feel
```

### Option 2: Progress Communication
```
SVG Line: Draws from left to right with dash effect
Step Dots: Turn color as steps complete
Progress Bar: Animates width based on stage
Result: Clear understanding of progress
```

### Option 3: Orbital Physics
```
12 Particles: Orbit in perfect circles
Center Glow: Pulses in sync with particles
Gradient Text: Fades in/out with breathing effect
Underline: Scales in and out
Result: Maximum visual impact, premium feel
```

### Option 4: Smooth Efficiency
```
Icon: Continuous rotation (3s smooth spin)
Progress Bar: Smooth width animation (2s cycle)
Indicator Dot: Gentle pulse (1.5s cycle)
Result: Professional, compact, performant
```

  const stageMap = {
    thinking: 'analyzing',
    generating: 'generating',
    chart: 'processing'
  };
  return <HybridSmartLoader stage={stageMap[stage] || 'analyzing'} />;
};
